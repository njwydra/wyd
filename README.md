# wyd

A barebones web app built with Vite, TypeScript, HTML, and CSS.

## Setup

Install dependencies:

```bash
npm install
```

## Development

Start the development server with hot module reload:

```bash
npm run dev
```

Then open your browser to `http://localhost:5173`

## Build

Build for production:

```bash
npm run build
```

The compiled assets will be in the `dist/` folder.

## Preview

Preview the production build locally:

```bash
npm run preview
```
